Changelog 1.2.1

    -the explorer forces the game to "Remember login" and "Automatically loging".
     This is to avoid that when changing the server map the game does not connect
     to the new map due to a blank username error.
    -the shoutbox load was changed so that it loads after sending requests to the servers.   
     this to prevent possible interference in the sending of udp packages.
    -fixed the problem of closing an AA server previously opened when closing the browser.
    -Added help file in korean thanks to UtE
       

Changelog 1.2

    -Added Dutch, German and Korean language
    -full support for ddns (for those who want to run an online server with dynamic IP)
    -added the option to run a server in the CMD
    -the browser now save the last server setup
    -Can detect and list "LAN servers" created with the browser or the traditional way
    -the waiting time was reduced to 10 seconds to load the backup list, when the chevy website does not respond.
    -Added a Favorites Tab (to add servers that do not appear in the list)
    -added the option to configure the screen resolution of the game
    -the shout box was modified to show a clock with a reference time
